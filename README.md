### Тема: Проєктування та реалізація клієнт-серверного мобільного додатку для підтримки та організації Настільних Рольових Ігор, що забезпечує автоматичне керування ігровими правилами та створення користувацького контенту

### Автор: Соколов Ілля Костянтинович

### Науковий керівник:  Заяць Марта Петрівна, UX/UI | Product Designer

### Назва: Aether

### Опис проєкту
Цей проєкт є кросплатформним мобільним застосунком, розробленим для автоматизації рутинних процесів, ведення менеджменту контенту та координації ігрових сесій у настільних рольових іграх (на прикладі системи Dungeons & Dragons 5e). Застосунок реалізовано за триланковою архітектурною моделлю (Client-Server-Database) із чітким розмежуванням логіки.

**Основний стек технологій:**
* **Frontend:** React Native, Expo, Expo Router (клієнтська маршрутизація).
* **Backend:** Node.js, Express.js (серверні роутери для обробки REST API запитів).
* **Бази даних та сховища:** MongoDB (NoSQL), Supabase Storage (Object Storage для медіа-файлів).
* **Інтеграції:** D&D 5e API (зовнішні довідники правил) та спроектований модуль аналітики на базі LLM Llama.
